Sublista contígua de soma máxima

Um problema clássico da Ciência da Computação é encontrar a sublísta contígua de soma máxima, ou o maior subarranjo, ou subvetor de soma máxima a partir de uma lista ou vetor de números.

No arranjo acima temos a soma máxima de //STUB começando na posição %d e terminando na posição %d. É importante notar que geralmente o problema envolve um conjunto de números inteiros, com elementos positivos e negativos, já que, se não houver números negativos, toda a lista é uma solução trivial para o problema.

Aplicações

1. simplified model for maximum likelihood estimation of patterns in digitized images.
2. Maximum subarray problem adapted for manycore architectures

void seg_max(int *v, int n, int & x, int &y , int & max){
int i,j;
int soma; 
max = -1; 
for( i = 0; i < n; ++i){
	soma = 0;
	for( j = i; j < n; ++j){

}

Algoritmo de Kadane

int kadane(int *v, int n, int & x, int &y){
int max_atual, max_total;
int temp;
int i;
max_atual = 0;
max_total = -1;
temp = 0;
for( i=0; i < n; ++i){
 	max_atual = max_atual + v[i];
	if(max_atual < 0) { 
		max_atual = 0;
		temp = i+1;
  	}
  	if(max_atual > max_total){
   		max_total = max_atual;
   		x = temp;
   		y = i;
  	} 
}
return max_total;
}
