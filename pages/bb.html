O que é busca binária?

É um dos algoritmos fundamentais da ciência da computação e é utilizado para realizar pesquisas em vetores ordenados. Tal algoritmo possui complexidade de tempo de execução igual a Ο(log n).

O que significa ser O(log n)?

Ao se criar um algoritmo, um dos pontos a se verificar é a complexidade temporal do programa, pois a partir dela pode-se vericar a eficiência do algoritmo. De forma geral, a complexidade se baseia nos loops (estruturas de repetição) que o programa utiliza.

Um caso fácil para analisar é a impressão de todos os números de um vetor. Como o laço desse programa irá percorrer todos os elementos do vetor, então dizemos que esse loop executa n vezes. Como essa é a única estrutura de repetição do algoritmo, diz-se que o programa tem complexidade temporal de O(n). Então, na busca binária, o laço do algoritmo sempre está trabalhando com um vetor de n/2, ou seja, a cada execução o vetor diminui seu tamanho em n/2 e esse tipo de execução caracteriza-se pelo log n, então o algoritmo possui complexidade de O(log n). Isso significa que a utilização da busca binária é melhor que usar uma busca linear, já que O(log n) < O(n).

Princípio de funcionamento

Tomemos o seguinte vetor como exemplo, onde desejamos encontrar o valor 31.

Então, estabelecemos 3 índices:

  • esquerdo = 0
  • direito = 9
  • meio = (direito + esquerdo) / 2 = 4

O algoritmo sempre irá verificar o valor na posição meio, de modo que se o valor procurado estiver nessa posição, o algoritmo é finalizado, retornando o índice meio. Caso o valor na posição meio seja > que o procurado, então direito = meio - 1, caso contrário, esquerdo = meio + 1. Então o algoritmo é executado até que o valor desejado seja encontrado, ou quando esquerdo > direito, neste caso o algoritmo retorna -1, já que não o valor desejado não está na lista. Continuando o exemplo acima, temos a seguinte situação:

É comparada o valor desejado, 31, com o valor na posição 4, 27. Como vetor[meio] < 31, então, esquerdo = meio + 1. Nesse momento, um novo meio é calculado, seguindo a mesma equação definida anteriormente, ficando com o valor 7.

É repetido o mesmo processo anterior, onde compara-se o valor de vetor[meio] com 31, e nesse caso vetor[meio] > 31, então direito = meio - 1. Um novo meio é cálculado, ficando com valor 5.

Nesse momento, ao fazer a comparação de vetor[meio] com 31, tem-se que vetor[meio] == 31, então o algoritmo retorna o valor 5 e finaliza o algoritmo.

Algoritmos

Busca binária convencional

int bb(int tamanho_vetor, int v[], int valor){
    int esq = 0; // Índice esquerdo
    int dir = tamanho_vetor - 1; // Índice direito
    int meio; // Índice meio
    while(esq <= dir){ // Condição de parada
        meio = (esq + dir) / 2; // Cálculo do meio
        if(v[meio] == valor) // Verifica se o v na posição meio é igual ao valor procurado
            return meio; // Retorna a posição caso for encontrada
        else if(v[meio] > valor) // Caso v na posição meio seja diferente do valor procurado, verifica-se v nessa posição é maior que o valor.
            dir = meio - 1; // Atualiza o valor do índice dir
        else // Caso contrário
            esq = meio + 1; // Atualiza o valor do índice esq
    }
    return -1; // Condição em que a busca falhou e o elemento não foi encontrado.
}
Busca binária - Retorna o primeiro elemento que seja maior ou igual que o valor

int lowerBB(int tamanho_vetor, int v[], int valor){
    int esq = 0; // Índice esquerdo
    int dir = tamanho_vetor - 1; // Índice direito
    int meio; // Índice meio
    while(dir - esq > 1){ // Condição de parada
        meio = (esq + dir) / 2; // Cálculo do meio
        if(v[meio] == valor) // Verifica se o v na posição meio é igual ao valor procurado
            return meio; // Retorna a posição caso for encontrada
        else if(v[meio] > valor) // Caso v na posição meio seja diferente do valor procurado, verifica-se v nessa posição é maior que o valor.
            dir = meio; // Atualiza o valor do índice dir
        else // Caso contrário
            esq = meio; // Atualiza o valor do índice esq
    }
    if(v[esq] >= valor) return esq; //Caso v[esq] >= valor, retorna-se esq
    else if(v[dir] >= valor) return dir; //Caso v[dir] >= valor, retorna-se esq
    else return -1; // Caso não exista algum elemento dentro de 'v' que seja maior que valor, retorna-se -1
}

int upperBB(int tamanho_vetor, int v[], int valor){
    int esq = 0; // Índice esquerdo
    int dir = tamanho_vetor - 1; // Índice direito
    int meio; // Índice meio
    while(dir - esq > 1){ // Condição de parada
        meio = (esq + dir) / 2; // Cálculo do meio
        if(v[meio] == valor) // Verifica se o v na posição meio é igual ao valor procurado
            return meio; // Retorna a posição caso for encontrada
        else if(v[meio] > valor) // Caso v na posição meio seja diferente do valor procurado, verifica-se v nessa posição é maior que o valor.
            dir = meio; // Atualiza o valor do índice dir
        else // Caso contrário
            esq = meio; // Atualiza o valor do índice esq
    }

    if(v[dir] <= valor) return dir; //Caso v[dir] <= valor, retorna-se dir
    else if(v[esq] <= valor) return esq; //Caso v[esq] <= valor, retorna-se esq
    else return -1; // Caso não exista algum elemento dentro de 'v' que seja menor que valor, retorna-se -1
}
